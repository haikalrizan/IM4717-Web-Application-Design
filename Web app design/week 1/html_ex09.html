TCP
Transmission Control Protocol is a method (protocol) used along with the Internet Protocol (IP) to send data in the form of message units, called packets, between computers over the internet.
IP
Internet Protocol is the method or protocol by which data is sent form one computer to another on the internet. Each computer on the Internet is uniqeluy identified by an IP address.
